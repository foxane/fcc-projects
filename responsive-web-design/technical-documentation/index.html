Introduction

Responsive web design is an approach to make web pages render well on variety of devices.

Before you write your first CSS line, HTML are responsive by default. It may not look good, but it isresponsive. So, when you styling your page, and suddenly something broke, know that it is your fault. But don't worry, I am here to help you not wasting time scracthing your head. At least not as much as me.

The rest of this page will show what's on your arsenal that you might not know.

Media Query

Media queries enable you to apply CSS styles based on a device's media type (e.g., print or screen) or its various attributes, including screen resolution, orientation, aspect ratio, browser viewport width or height, and user preferences like preferring reduced motion, data usage, or transparency.

Below example will apply color: red; into .container when the screen width are 600px or above.

                    
                        @media (min-width: 600px) {
                            .container {
                                color: red;
                            }
                        }

In your stylesheet, you can include different media queries to adjust your layout or parts of it for various screen sizes. These adjustments happen at specific points called breakpoints.

Use your breakpoints when the page start to look bad at certain screen width.
